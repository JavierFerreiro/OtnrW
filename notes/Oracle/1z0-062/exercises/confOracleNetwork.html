1-Configure Oracle Net

  1. Create a directory to be used for Oracle Net only for this test.
    mkdir /home/oracle/netTest
    export TNS_ADMIN=/home/oracle/netTest
    --To check if all is correct:
    tnsping orclscripts
  2. Start the net manager and create a new listener
    netmgr
    #Be sure that in the title of the app the path are the previously defined in TNS_ADMIN
    #Listener name: NEWLIST
    #Port Listener: 2521
    #IP:127.0.0.1
  3. Create a new service name
    #Service name: SERV1
    #Port Listener: 2521
    #IP:127.0.0.1
  4. Start the new listener
    lsnrctl start newlist
  5. Set the DB to use the new listener/service
    1. sqlplus / as sysdba
    2. alter system set service_name=serv1 scope=memory;
    3. alter system set local_listener=new scope=memory;
    4. alter system register;
  6. Check from the OS that the service are register in the listener and connect to them.
    1. lsnrctl status newlist
    2. sqlplus system@new
